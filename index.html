Mac, all of these are cool but what are you currently working on?

Well, I'm, glad you asked! at the moment I'm working with a group of peers to extend and improve a semester long class project that we worked on over the last term. This project is a course planning tool that University of Guelph students can use to plan their semesters out.

Of course Guelph already has a tool for planning courses but it's pretty lackluster. This tool will provide new functionalities and make it easier for students to get the courses they want when they want them.
